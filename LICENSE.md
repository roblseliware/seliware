## 🛠️ Common Issues & Fixes

### ❌ "Injection Failed" Error
**Cause:** Roblox was updated and the current Seliware version hasn't been patched yet.
**Fix:** Check the official Discord for an update announcement. Wait for the patch (usually within 24–48 hours) or roll back to the previous Roblox version using a version manager.

### ❌ Antivirus Deletes Seliware on Download
**Cause:** Executor injection methods trigger heuristic AV detection (false positive).
**Fix:** Add the Seliware folder to your antivirus exclusion/whitelist list before downloading. This is standard practice for all script executors.

### ❌ Roblox Crashes Immediately After Injection
**Cause:** Incompatible script or injection triggered during loading screen.
**Fix:** Always wait until you are fully inside a game server before injecting. If crashes persist, try a clean reinstall of Seliware.

### ❌ Scripts Execute But Do Nothing
**Cause:** The script may be outdated or game-specific (patched by the game developer).
**Fix:** Download an updated version of the script from a trusted source. Check the built-in Script Hub for maintained alternatives.

### ❌ "Key Not Valid" Error on Launch
**Cause:** The 24-hour key has expired or was entered incorrectly.
**Fix:** Generate a new key from the official Seliware key system page. Keys reset every 24 hours on the free tier.

### ❌ Seliware UI Not Appearing After Injection
**Cause:** UI process failed to initialize, often due to overlay software conflicts (Discord overlay, GeForce Experience, etc.).

### ❌ Android Build: "App Not Installed"
